# CodeAlpha_Calculator
🧮 Stylish Web Calculator

A modern web-based calculator built using HTML, CSS, and JavaScript as part of my CodeAlpha Internship Program.
This project focuses on creating an interactive UI and implementing calculator logic using JavaScript.

🚀 Features

✨ Clean and modern calculator interface
➕ Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
📊 Percentage calculation support
🧹 Clear (AC) and Delete (DEL) functionality
🎨 Attractive gradient UI with smooth button effects
⚡ Lightweight and runs directly in the browser

🛠️ Technologies Used

🌐 HTML5 – Structure of the calculator

🎨 CSS3 – Styling and responsive UI design

⚡ JavaScript – Calculator logic and event handling
